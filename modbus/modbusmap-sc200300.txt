SC300 Modbus Server v6.0				
The SC300 implements Modbus/TCP Server as described in Modbus Messaging On TCP/IP Implementation Guide V1.0b section 4.4.2.				
It accepts connections on port 502				
The SC300 implements Modbus/RTU in accordance with MODBUS over Serial V1.02 section 2 (Data Link Layer) to achieve a compliance level of "conditionally compliant". This means the SC300 modbus server				
It auto-detects Modbus/RTU communications on the RS-232 port				
The SC300 does not support Modbus/ASCII				
The SC300 implements the following Modbus functions according to the format described in Modbus Application Protocol Specification V1.1b.
